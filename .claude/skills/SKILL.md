# JOBISEO Claude Skill Pack

This folder contains lightweight skill prompts for disciplined repository work.

## Skill Index

- `jobiseo_init.skill.yaml` — bootstrap a session and align with workspace rules
- `workspace_docs.skill.yaml` — edit identity and operations documents consistently
- `repo_hygiene.skill.yaml` — keep public repo safety and git hygiene intact
- `verification_agent.skill.yaml` — review diffs, run checks, and report residual risk

## Usage Pattern

1. Read the relevant skill file.
2. Inspect current files before changing them.
3. Apply the smallest stable edit.
4. Verify the result.
5. Summarize what changed and what remains uncertain.
